## 5. 종합 분석 및 다음 실험 방향

### 현재까지의 결론

1. **`coc` 모드 >> `spatial` 모드**: coc가 모든 평면에서 일관되게 우수 (최대 25.9 dB vs 최대 21.5 dB).
2. **Step Size = 0.2가 적합**: 0.002는 모든 실험에서 실패.
3. **GM Steps 50 vs 100**: 50 스텝에서도 충분한 성능. 100 스텝이 항상 더 좋지는 않음.
4. **Eta Schedule**: `constant`가 single scene에서 최고 피크를 찍었지만, `cosine`/`linear`는 **Latest 모델이 Best보다 좋은 패턴**을 보여줌 → val_loss 기반 best 판정이 PSNR과 불일치할 가능성. 이 스케줄들의 장기 수렴 성능은 긍정적.
5. **`conv1x1` 헤드**: single scene + constant에서 발산. multi-scene + cosine에서는 의외로 fc와 비슷한 수준 (20~21 dB). 추가 검증 필요.
6. **LR**: 1e-4가 기본. 3e-5는 일부 안정적(#8: 23.6 dB)이나 일부 불안(#7: latest 하락).
7. **Multi-Scene 확장**: #19, #20에서 5장으로 스케일업 시작. 학습 완료 시 성능 확인 필요.

### 주의사항

- **Best 모델 ≠ 최고 PSNR일 수 있음**: `train.py`의 best 판정은 `val_loss` 기준이므로, PSNR과 완전히 일치하지 않을 수 있음. 특히 cosine/linear 스케줄 실험(#11, #12)에서 이 현상이 두드러짐.
- **#9 과적합**: Best(ep39)=25.9 dB → Latest(ep100)=23.4 dB. 100 에폭은 single scene에서 과적합 위험.
